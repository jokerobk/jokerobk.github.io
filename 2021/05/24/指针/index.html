    空指针和野指针

  • 空指针

    指针变量指向内存中编号为0的空间

  • 空指针的用途

    初始化指针变量

  • 注意

    空指针指向的内存是不可以访问的

    #include<iostream>
using namespace std;
int main(){
  int *p = NULL;  //空指针
  // *p = 1； 错误操作
  return 0;
}
  • 野指针

    指针变量指向非法的内存空间,在程序中要避免出现野指针

    int *a = (int *)0x3f3f;     //野指针
  • 总结

    空指针和野指针都不是我们申请的空间，因此不要访问

const修饰指针

  • const修饰指针的三种情况
    1. const修饰指针——常量指针
    2. Const修饰常量——指针常量
    3. const即修饰指针，又修饰常量
  • 常量指针
    int a = 10;
int b = 10;
const int *p =&a;

    特点：指针的指向可以修改，但是指针指向的值不可以改。
    *p = 20;(错误，指针指向的值不可以改)
    p = &b;(正确，指针指向可以修改)

  • 指针常量
    int a = 10;
int b = 10;
int * const p = &a;

    特点：指针的指向不可以修改，但是指针指向的值可以修改。
    *p = 20；(正确，指针指向的值可以改)
    p = &b;(错误，指针的指向不可以改)

  • 即修饰指针，又修饰常量
    int a = 10;
int b = 10;
const int * const p = &a;

    特点：指针的指向既不可以修改，指针指向的值也不可修改。

指针和数组

&#160将指针指向数组和首地址，例如有一个整形数组a[10],数组a的首地址即为a，可将p=a;就表示指针p指向了数组a的首地址，可通过p++或p+1来访问数组a中后面的元素。当指针p指向数组a的首地址时，p中存放的为a[0]的地址，通过解引用使用p就可以访问a[0]中的元素，p++就是将指针p后移4个字节，使p指向a[1];而p+n(n为整数，可为正，也可为负)就表示p中存放的地址加上n*4个字节，例如p+1指a[1],p+2也就指向a[2]。

  • code
    #include <iostream>
#include <cstring>
using namespace std;
int main()
{
  int a[10];
  int *p = a;
  for (int i = 0; i < 10; i++)
  {
      a[i] = i;
      cout << *(p + i) << " ";
  }
  cout << endl;
  memset(a, 0, sizeof(a));
  for (int i = 0; i < 10; i++)
  {
      *p = i;
      p++;
      cout << a[i] << " ";
  }
  p--;
  return 0;
}

指针和函数

  • 函数值传递
    void swap1(int a, int b){
  int temp = a;
  a = b;
  b = temp;
}
int x = 10,y=20;
swap1(x,y);

    不会改变实参

  • 地址传递
    void swap2(int *a, int *b)
{
  int temp = *a;
  *a = *b;
  *b = temp;
}
Int x = 10, y = 20;
swap2(&x, &y);

    这种地址传递方式会改变实参x，y的值，但不会改变a和b的指向，指针a还是指向x，指针b还是指向y。

    void swap3(int *a, int *b)
{
  int *temp = a;
  a = b;
  b = temp;
}
Int x = 10, y = 20;
swap3(x, y);

    这种地址传递方式不会改变实参x，y的值，但会改变指针a和b的指向，指针a指向y，指针b指向x。
