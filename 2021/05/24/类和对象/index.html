C++封装

c++面向对象的三大特性为：封装、继承、多态

C++类定义

类定义是以关键字class开头，后跟类的名称，累的主体是包含在一对花括号中。类定义后必须跟着一个分号或者一个生命列表。例如，使用关键字class定义Box数据类型，如下所示：

class Box{
    public:
        double length;        //盒子的长度
        double breadth;        //盒子的宽度
        double height;        //盒子的高度
};

关键字public确定了类成员的访问属性。在类对象作用域内，公共成员在类的外部都是可以访问的。类中默认的构造函数的函数体为空。也可以在类中写出该类的构造函数，如下所示：

class R : public Shape
{
public:
    R(int width, int height)
    {
        this->width = width;
        this->height = height;
    }
    int getArea()
    {
        return width * height;
    }
};

C++对象定义

类提供了对象的蓝图，基本上，对象是根据类来创建的。声明类的对象，就像声明基本类型的变量一样。如下：

Box box1;
Box box2;

访问数据成员

类的对象的公共数据成员可以使用直接成员访问运算符“.”访问。实例如下：

#include <iostream>
using namespace std;
class Box
{
public:
    double length;
    double breadth;
    double height;
    //成员函数声明
    double get(void);
    void set(double len, double bre, double hei);
};
//
double Box::get(void)
{
    return length * breadth * height;
}
void Box::set(double length, double breadth, double height)
{
    this->length = length;
    this->breadth = breadth;
    this->height = height;
}
int main()
{
    Box box1;
    Box box2;
    Box box3;
    double volume = 0.0;
    //box1信息
    box1.length = 6.0;
    box1.breadth = 7.0;
    box1.height = 5.0;
    //box2信息
    box2.length = 12.0;
    box2.breadth = 13.0;
    box2.height = 10.0;
    //box1体积
    volume = box1.length * box1.breadth * box1.height;
    cout << "box1的体积: " << volume << endl;
    //box2体积
    volume = box2.length * box2.breadth * box2.height;
    cout << "box2的体积: " << volume << endl;
    //box3信息
    box3.set(16.0, 8.0, 12.0);
    volume = box3.get();
    cout << "box3的体积: " << volume << endl;
    return 0;
}

指向类的指针

一个指向C++类的指针与指向结构体的指针类似，访问指向类的指针的成员，需要使用成员访问运算符**->**，就像访问指向结构体的指针一样。与所有的指针一样，必须在使用指针之前，对指针进行初始化。实例如下：

#include <iostream>
using namespace std;
class Box
{
public:
 //构造函数
 Box(double l = 2.0, double b = 2.0, double h = 2.0)
 {
     cout << "Constructor called." << endl;
     length = l;
     breadth = b;
     height = h;
 }
 double Volume()
 {
     return length * breadth * height;
 }

private:
 double length;
 double breadth;
 double height;
};

int main()
{
 Box box1(3.3, 1.2, 1.5);
 Box box2(8.5, 6.0, 2.0);
 Box *ptrBox;
 //保存box1的地址
 ptrBox = &box1;
 cout << "Volume of box1: " << ptrBox->Volume() << endl;
 ptrBox = &box2;
 cout << "Volume of box2: " << ptrBox->Volume() << endl;
 return 0;
}

也可以通过另一种方式创建类的对象：

class student{
  string name;
  int age;
};
student *pStu = new student;

通过new创建出来的对象，它在堆上分配内存，没有名字，只能得到一个指向它的指针，所以必须使用一个指针变量来接收这个指针，否则以后再也无法找到这个对象了，更没有办法使用它。使用new在堆上创建出来的对象是匿名的，必须要用一个指针指向它，在借助指针来访问它的成员变量或成员函数。

栈内存是程序自动管理的，不能使用delete删除在栈上创建的对象；堆内存由程序员管理，对象使用完毕后可以通过delete删除。在实际开发中，new和delete往往成对出现，以保证及时删除不再使用的对象，防止误用内存堆积。实例如下:

#include <iostream>
#include <cstring>
using namespace std;
class Student
{
public:
    string name;
    int age;
    int score;

    void say()
    {
        cout << name << "的年龄是" << age << "岁，成绩是" << score << endl;
    }
};

int main()
{
    Student *pStu = new Student;
    pStu->name = "小明";
    pStu->age = 19;
    pStu->score = 100;
    pStu->say();
    delete pStu;
    return 0;
}

构造函数和析构函数

  • 构造函数

主要作用在创建对象时为对象的成员属性赋值，构造函数由编译器自动调用，无须手动调用。

类名(){}

可以有参数，可以重载

  • 析构函数

主要作用在对象销毁前系统自动调用，执行一些清理工作。

～类名(){}

不能有参数，不可以重载

构造函数的分类及调用

  • 构造函数有两种分类方式：

按参数分为：有参构造和无参构造

按类型分为：普通构造和拷贝构造

  • 三种调用方式：

括号法、显示法、隐式转换法

person(const person p);

将传入的对象身上的所有属性，拷贝到调用对象的身上

person::person(const person &p)

构造函数调用规则

c++在默认情况下会给一个类默认创建默认构造函数、默认析构函数、默认拷贝函数。

如果用户定义有参构造函数，则不再默认提供默认无参构造，但仍有默认析构和默认拷贝。

如果用户定义拷贝构造函数，则不会提供另外的默认构造函数。

默认的拷贝构造函数会对传进去的对象的成员属性进行复制，并赋值给当前对象。

深拷贝和浅拷贝

  • 浅拷贝

简单的赋值拷贝操作

浅拷贝会导致堆区的内存重复释放

需要自己写一个拷贝构造函数解决浅拷贝带来的问题

  • 深拷贝

在堆区重新申请空间，进行拷贝操作

  • 深拷贝示例代码
#include <iostream>
using namespace std;
class person
{
public:
    person()
    {
        cout << "默认构造" << endl;
    }
    person(int age, int height)
    {
        m_age = age;
        m_height = new int(height);
        cout << "有参构造" << endl;
    }
    ~person()
    {
        if (m_height != NULL)
        {
            delete m_height;
            m_height = NULL;
        }
        cout << "析构函数" << endl;
    }
    person(const person &p)
    {
        cout << "拷贝构造" << endl;
        m_height = new int(*p.m_height);
        m_age = p.m_age;
    }
    int m_age;
    int *m_height;
};
void demo1()
{
    person p1(18, 180);
    cout << "p1身高: " << *p1.m_height << "p1年龄: " << p1.m_age << endl;
    person p2(p1);
    cout << "p2身高: " << *p2.m_height << "p2年龄: " << p2.m_age << endl;
}
int main()
{
    demo1();
    return 0;
}
如果属性中在堆区开辟的，一定要自己写拷贝构造函数。

静态成员

在类中由static关键字修饰的成员函数称为静态成员函数，由static修饰的成员变量称为静态成员变量。

所有对象共享一个同一个静态成员函数。

静态成员函数只能访问静态成员变量。

#include <iostream>
using namespace std;
class person
{
public:
    static void func()
    {
        m_a = 100;
        //m_b = 100;    无法访问
        cout << "静态成员函数" << endl;
        cout << m_a << endl;
    }
    static int m_a;
    int m_b;
};

int person::m_a = 10;

int main()
{
    person p1;
    p1.func();
    person::func();
    return 0;
}

C++继承

继承

面向对象程序设计中最重要的一个概念是继承。继承允许我们依据另一个类来定义一个类，这使得创建和维护一个应用程序变得更容易。也达到了重用代码功能和提高执行效率的效果。

当创建一个类时，不需要重新编写新的数据成员和成员函数，只需指定新建的类继承了一个已有的类的成员即可。这个已有的类称为基类，新建的类称为派生类。

继承代表了b is a关系。例如狗是动物等。实例如下：

// 基类
class Animal{
    void eat();
    void sleep();
};
// 派生类
class Dog : public Animal{
    void bark();
};

基类&派生类

一个类可以派生自多个类，这意味着它可以从多个基类继承数据和函数。定义一个派生类，使用一个类派生列表来指定基类。类派生列表绎一个或多个基类命名。形式如下：

class 派生类名 : access-specifier 基类名
其中访问修饰符access-specifier是public、private或protected中的一个。如果未使用访问修饰符，则默认使用private。
  • 实例代码
#include <iostream>
using namespace std;
class Shape
{
public:
 void setwidth(int width)
 {
     this->width = width;
 }
 void setheight(int height)
 {
     this->height = height;
 }

protected:
 int width;
 int height;
};

class R : public Shape
{
public:
 int getArea()
 {
     return width * height;
 }
};

int main()
{
 R r;
 r.setwidth(10);
 r.setheight(5);
 cout << "Area:" << r.getArea() << endl;
 return 0;
}
  • 输出结果
Area:50

访问控制和继承

派生类可以访问基类中所有的非私有成员。

访问 public protected private
同一个类 yes yes yes
派生类 yes yes no
外部的类 yes no no

一个派生类继承了所有的基类方法，但不包括以下几种情况：

  1. 基类的构造函数、析构函数和拷贝构造函数。
  2. 基类的重载运算符。
  3. 基类的友元函数。

多继承

派生类可以继承多个基类，继承多个基类的特性，实例如下：

#include <iostream>
using namespace std;
class Shape
{
public:
    void setwidth(int width)
    {
        this->width = width;
    }
    void setheight(int height)
    {
        this->height = height;
    }

protected:
    int width;
    int height;
};

class price
{
public:
    int getMoney(int area)
    {
        return area * 100;
    }
};

class R : public Shape, public price
{
public:
    R(int width, int height)
    {
        this->width = width;
        this->height = height;
    }
    int getArea()
    {
        return width * height;
    }
};

int main()
{
    R r(10, 5);
    //r.setwidth(10);
    //r.setheight(5);
    cout << "Area: " << r.getArea() << endl;
    cout << "Money: " << r.getMoney(r.getArea()) << endl;
    return 0;
}

继承方式

子类继承父类后，子类无法访问父类的private属性及方法，当子类继承不同权限的父类后，父类中的成员到子类后的的权限变更如下表，仅针对父类中的public权限和protected权限。

继承 public protected private
子类 保持不变 除private全部变为protected 全部变为private 
#include<iostream>
#include<cstring>
using namespace std;
class father{
    public:
    int m_A;
    protected:
    int m_B;
    private:
    int m_C;
};
class son1 : public father{
    public:
    void func(){
        m_A = 10;            //权限保持不变
        m_B = 10;            //权限保持不变
        //m_C = 10;   //无法访问。
    }
};
class son2 : protected father{
    public:
    void func(){
        m_A = 10;            //权限变为protected
        m_B = 10;            //权限保持不变
    }
};
class son3 : private father{
    public:
    void func(){
        m_A = 10;            //权限变为private
        m_B = 10;            //权限变为private
    }
};
void test01(){
    son1 s1;
    s1.m_A = 20;//只能访问父类中的public
    son2 s2;
    //s2.m_A = 10;  //无法访问，因为son2保护继承father，
    //father中的public到son2中变为了protected
    son3 s3;
    //s3.m_A = 10;  无法访问，son3私有继承father，
    //father中的public和protected到son3中变为了private，无法访问。
}

继承中构造和析构顺序

当创建子类对象时，会先调用父类构造，再调用子类构造，析构为先调用子类的析构再调用父类的析构，并且会创建一个父类对象。

#include <iostream>
using namespace std;
class father
{
public:
    father()
    {
        cout << "father构造函数" << endl;
    }
    ~father()
    {
        cout << "father析构函数" << endl;
    }
};
class son : public father
{
public:
    son()
    {
        cout << "son构造函数" << endl;
    }
    ~son()
    {
        cout << "son析构函数" << endl;
    }
};
void test()
{
    son s;
}
int main()
{
    test();
    return 0;
}

继承同名成员处理方式

如果子类和父类有同名的成员属性或成员方法，父类中的同名成员函数会被隐藏，要想通过子类对象调用父类中同名成员属性或成员方法，需要在成员属性或成员方法前父类的域解析符。

#include <iostream>
using namespace std;
class father
{
public:
    int m_A;
    father()
    {
        m_A = 10;
    }
    void func()
    {
        cout << "father" << endl;
    }
};
class son : public father
{
public:
    int m_A;
    son()
    {
        m_A = 20;
    }
    void func()
    {
        cout << "son" << endl;
    }
};
void test01()
{
    son s;
    cout << s.m_A << endl;
    //通过子类访问父类中的同名成员属性，需要加父类的域解析符
    cout << s.father::m_A << endl;
}
void test02()
{
    son s;
    s.func();
    s.father::func();
}
int main()
{
    test01();
    test02();
    return 0;
}

多态

多态的基本概念

  • 静态多态

函数重载和运算符重载属于静态多态，复用函数名。

  • 动态多态

派生类和虚函数实现运行时多态。

  • 静态多态和动态多态的区别

静态多态的函数地址早绑定—编译阶段确定函数地址

动态多态的函数地址晚绑定—运行阶段确定函数地址

  • 虚函数

在函数名前面加上virtual变成虚函数，在编译时才确定函数地址

  • 动态多态满足条件
  1. 有继承关系
  2. 派生类重写基类的虚函数
  3. 基类的指针或引用指向派生类对象
  • 实例代码
#include <iostream>
using namespace std;
class Animal
{
public:
    virtual void speak()
    { //函数前面加上virtual变成
        cout << "动物会叫" << endl;
    }
};
class Cat : public Animal
{
public:
    void speak()
    {
        cout << "喵喵" << endl;
    }
};
void Speak(Animal &a)
{
    a.speak();
}
void test()
{
    Cat cat;
    Speak(cat);
}
int main()
{
    test();
    return 0;
}
  • 虚函数底层

虚函数底层

纯虚函数和抽象类

  • 纯虚函数语法

virtual 返回值类型 函数名 (参数列表) = 0；

当类中有了纯虚函数，这个类也称为抽象类

  • 抽象类特点

无法实例化对象

派生类必须重写抽象类中的纯虚函数，否则也属于抽象类

#include <iostream>
using namespace std;
class Animal
{
public:
    virtual void speak() = 0;
};
class Cat : public Animal
{
    void speak()
    {
        cout << "喵喵" << endl;
    }
};

class Dog : public Animal
{
    void speak()
    {
        cout << "汪汪汪" << endl;
    }
};
void test()
{
    Animal *a = new Cat; //无法使用引用，只能使用指针
    a->speak();
    delete a;
}
int main()
{
    test();
    return 0;
}
