引用

引用的本质是一个指针常量

引用的基本使用

引用就相当于给变量重新起了一个别名，数据类型要一致。

int a = 10;
int &b = a;

对a进行操作b会改变，对b进行操作a也会改变。

引用注意事项

引用必须要初始化，且初始化后不可改变。

引用做函数参数

函数传参时，可以利用引用的技术让形参修饰实参，此时形参是实参的一个别名，当形参改变时，实参也会改变。

引用做函数返回值

引用是可以作为函数的返回值存在的，但不能返回局部变量的引用

函数的引用可以作为左值

#include <iostream>
using namespace std;
int &test02()
{
    static int a = 10;
    return a;
}
int main()
{
    int &ref = test02();
    cout << "ref=" << ref << endl;
    cout << "ref=" << ref << endl;
    test02() = 100;
    cout << "ref=" << ref << endl;
    return 0;
}

常量引用

用来修饰形参，防止在函数中误操作。

#include <iostream>
using namespace std;

void print(const int &val)
{
    //val = 1000;   无法对引用进行改变
    cout << "val=" << val << endl;
}

int main()
{
    int a = 100;
    print(a);
    return 0;
}
